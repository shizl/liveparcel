Please follow these instructions to attach views and rule to be imported during module instal.
1- Export Views settings into a .view file
2- Edit the .file to include <?php ?> tag
3- Place the file in /views folder
4- Repeat this for all the views you have

Instuctions for Rules
1- Export Rules settings into a .rule file
2- Edit the .rule file to include <?php $str='export_code';?>
3- export_code if the content initially output
4- repeat this for all the rule you have
